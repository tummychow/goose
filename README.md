## configuration

configuration is by environment variables

- `GOOSE_PORT` server address, eg `:4567` for port 4567 on any address
- `GOOSE_BACKEND` the backend URI, eg `file:///tmp/goose`
- `GOOSE_TEST_FILE` for testing the file backend
